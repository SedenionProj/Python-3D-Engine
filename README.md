# Python-3D-Engine

## A one-file 3D graphics engine written from scratch

### controls :

- french keyboard : use **Q D Z S** keys to move

- use **ARROW** keys to rotate

### fuctions :
##### **shapes** :
- `putPixel()`
- `triangle()`
- `triangle3D()`
- `mesh()`
##### **screen** :
- `clear()`
- `draw()`
##### **objects** :
- `loadObj()`
- `translate()`
- `scale()`
